Смарт-контракт "Хранилище датасетов" 
Предназначен для хранения и управления большим объемом данных. Предоставляет пользователям возможности загрузки и использования данных с оплатой монетами.
Пользователи могут загружать данные в хранилище, указывая метаданные. Кроме того, пользователи получают определенное количество монет за использование своих данных.

Содержание контракта:
- Data – структура данных, содержащая информацию о загруженном датасете, включая ID, имя и значения.
- lastId - приватная переменная для хранения последнего использованного ID;
- data - массив для хранения загруженных данных;
- balances - маппинг для хранения балансов пользователей.

События, вызываемые при определенных действиях пользователя:
- DataAdded - вызывается при добавлении новых данных;
- Deposit - вызывается при пополнении баланса пользователя;
- Withdrawal - вызывается при выводе средств с баланса пользователя;
- DataRequest - вызывается при создании запроса на загрузку данных.

Функции для взаимодействия пользователя со смарт-контрактом:
- addDataRequest - функция для оставления запроса на загрузку данных. Принимает имя и значения датасета в качестве параметров. Не требует оплаты;
- addData - функция для добавления новых данных в контракт. Принимает имя и значения датасета в качестве параметров. Требует оплату в монетах;
- getDataCount - функция для получения количества загруженных данных. Не требует оплаты;
- addExternalData - функция для добавления данных из внешнего контракта. Принимает адрес внешнего контракта в качестве параметра. Требует оплату в монетах;
- getData - функция для получения данных по ID. Принимает ID датасета в качестве параметра. Не требует оплаты;

Функции для для управления балансами пользователей:
- deposit - функция для пополнения баланса пользователя. Не требует параметров. Оплата производится в эфире;
- withdraw - функция для вывода средств с баланса пользователя на кошелек. Принимает количество монет, которое нужно вывести, в качестве параметра;
- balanceOf - функция для проверки баланса контракта. Не требует параметров.
